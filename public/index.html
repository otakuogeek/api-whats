Conectar WhatsApp + ChatGPT

Esta pantalla te guía para dejar tu bot funcionando y validar si tu archivo .env está completo.

Estado de configuración

No se muestran secretos, solo si cada variable está configurada.

Paso 1: Crear API key de OpenAI

  1. Entra a platform.openai.com e inicia sesión.
  2. Ve a API keys en el panel de tu organización/proyecto.
  3. Haz clic en Create new secret key.
  4. Copia la clave una sola vez y guárdala en tu .env como OPENAI_API_KEY=....
  5. Opcional: define OPENAI_MODEL=gpt-4o y OPENAI_REASONING_EFFORT=medium.

Paso 2: Configurar WhatsApp en Meta

  1. En tu app de Meta, entra a WhatsApp > Configuración de la API.
  2. Configura el webhook con tu URL pública (ngrok) terminada en /webhook.
  3. Usa el mismo valor en Meta y en WHATSAPP_VERIFY_TOKEN.
  4. Suscribe el campo messages.
  5. Copia WHATSAPP_ACCESS_TOKEN y WHATSAPP_PHONE_NUMBER_ID a tu .env.

Paso 3: (Opcional) Persistencia con Redis

  • Si no configuras Redis, usa memoria RAM (se pierde al reiniciar).
  • Con Redis: REDIS_URL=redis://localhost:6379.
  • Controla historial con MEMORY_MAX_MESSAGES=5.
Abre esta página en http://localhost:3000. Tu webhook sigue en /webhook y envío manual en /send-text.
